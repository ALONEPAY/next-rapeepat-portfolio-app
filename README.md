# next-rapeepat-portfolio-app

Portfolio website ของ **นายระพีพัฒน์ เทียมผล**

## 🛠️ พัฒนาด้วย
- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **Google Fonts** (Inter, Prompt, Roboto Mono)

## 📄 หน้าเว็บไซต์
| หน้า | Route |
|------|-------|
| Home | `/` |
| About Me | `/about` |
| Skills | `/me/skills` |
| Projects | `/me/projects` |
| Gallery | `/me/gallery` |
| Contact | `/contact` |

## 🚀 วิธีรัน
```bash
npm install
npm run dev
```
